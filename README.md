# Express Middleware Authentication Demo

A simple Node.js + Express project demonstrating how middleware works
by protecting a secret page using password-based authentication.

## Tech Stack
- Node.js
- Express.js
- Body-parser

## Features
- Custom middleware for password validation
- Request-based authorization (no global state)
- POST form handling
- Secure routing

## How to Run
```bash
npm install
node index.js
